Finding Zeros and Newton's Method

Since high school, we are taught to find the zeros of functions. In other words, we want to solve the following equation.

$$f(x)=0 $$

More often than not, this cannot be done by hand. If we are interested in approximate answers, which occurs often in practical applications, we can use Newton's method. Newton's method takes a starting point \(x_0\) and attempts to find a zero using the function \(f\) and its derivative \(f'\) by iterating as follows,

$$x_{n+1}=x_n-\frac{f(x_n)}{f'(x_n)} . $$

There are problems if \(f'(x_n)=0\), the sequence encounters a fixed point, or the sequence enters a cycle. Generally speaking, if one starts close to a root, then you are guaranteed to converge (with quadratic convergence). We can also generalize this to complex valued functions \(f(z)\) by the same rule.

$$z_{n+1}=z_n-\frac{f(z_n)}{f'(z_n)}$$

Newton Basins

If you start close to a root, you are guaranteed to converge, but what if you are not near any root of the function? This is not a trivial question. Arthur Cayley was the first to realize the complexity of this question.

Take the function \(f(z)=z^3-1\). The three zeros of this function are \(1\), \(e^{i2\pi/3}\), and \(e^{i4\pi/3}\). For every point on the complex plane, we are going to color the point pink if it tends to \(1\) under Newton's method, red for \(e^{i2\pi/3}\), and green for \(e^{i4\pi/3}\). The image we get is a fractal.

This means that the overall shape of the figure appears the same no matter the magnification. If you were given a random magnification, you would not be able to discern your absolute location (or narrow it down in the slightest). We can also do something similar for the function \(f(z)=z^4-1\) and get the following image.

Polynomials of the form \(f(z)=z^n-1\) will always give the cyclic patterns with each root of unity having its own region separated by a complicated boundary. Other polynomials give even more interesting shapes. This site has some interesting examples. Here we have the fractal associated with the polynomial \((z^4-1)( z^2-(1+i)^2)\).

These fractals are also an example of chaotic behavior. This means that knowing what happens at a single point does not tell much about what happens in the small disk surrounding it. If I tell you the point \(p\) tends to \(q\), then I cannot make a prediction about what happens when I add a small perturbation \(p+\gamma\).

Code

Although these patterns are very complex (no pun intended), they are actually easy to generate in practice. 

    roots = (tuple of roots)

    colors = {rooti -> colori }
    grid = [matrix of gridpoints]
    image = [matrix same size as grid]
    while num_iter < max_iter:
        iterate every point
        num_iter += 1

    for every point in grid:
        if converge:
            image[point] = colors[root]
        else:
            image[point] = black

Lastly, feel free to try modifications to this process. For example, if you put a complex number \(c\) in the iteration \(f(x_{n+1})=x_n-c \cdot f(x_n)/f'(x_n)\), we can change the Newton basin. This is what happens for \(f(x)=x^3-1\) with \(c=(1/2)+(1/2) i\). One can also try transcendental functions, step-dependent changes, etc. The possibilities are endless.
